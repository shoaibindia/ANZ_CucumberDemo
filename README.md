# ANZ_CucumberDemo

This is demo project for ANZ Borrow calculator using Java Cucumber 

Clone github from my location - git clone https://github.com/shoaibindia/ANZ_CucumberDemo.git

Open Eclipse or any editor and import the maven project and wait till it loads and download the maven dependencies

Go to Testrunner.java run it

Refresh the page and check the report in target/cucumber-html-report.html
